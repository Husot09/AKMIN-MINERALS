# Akmin Industrial Minerals Website

Modern Next.js website for Akmin Industrial Minerals.

## Run locally

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Edit content

Most page copy, navigation labels, material details, contact information and reusable section content live in:

```text
lib/content.ts
```

The generated hero image is stored at:

```text
public/images/akmin-mineral-hero.png
```

## Pages

- Home
- About Akmin
- Materials
- Ceramic Solutions
- R&D and Innovation
- Sustainability
- Contact

## Notes

The contact form is ready for integration with an email service, CRM endpoint or form provider. It is currently a static front-end form.
